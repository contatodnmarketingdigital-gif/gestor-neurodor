# Painel do Gestor — Neurociência na Dor (Imersão)

Página estática hospedada via **GitHub Pages**.

URL final: **https://contatodnmarketingdigital-gif.github.io/gestor-neurodor**

## Conteúdo
- Período: 09/04/2026 a 28/04/2026 (20 dias)
- 4 alertas críticos no topo (pausar / escalar / fadiga / oportunidade)
- Ações 0–48h com cards de pausar/escalar/ajustar
- Tabela completa dos top 10 criativos com decisão tática
- Análise de fadiga do AD0001 (gráfico ROAS × CTR diário)
- Heatmap demográfico (idade × gênero)
- Plano de criativos novos (4 por semana × 4 semanas = 16 criativos)
- KPIs semanais com metas e ações se não bater

## Arquivos
- `index.html` — versão servida pelo GitHub Pages
- `Relatorio_Gestor_Neurodor.html` — cópia com nome descritivo
- `.nojekyll` — desliga processamento Jekyll

## Como atualizar
1. Abra o repositório no github.com
2. Clique em `Add file` → `Upload files`
3. Arraste o novo HTML por cima
4. Commit em `main`
5. A página atualiza em ~30 segundos
